## 🛠️ Install

```sh
yarn add shadows
```

## 🚗 Run

```sh
import { shadows } from "shadows-maker";

shadows({
    shadow_type: 'soft',
    padding: false
})
```

## 🔎 Options

* *shadow_type* - hard | soft (soft by default)
* *padding* - true | false (false by default)
